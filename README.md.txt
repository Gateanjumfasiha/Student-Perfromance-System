# Student Performance System

A Flask-based web application to manage student marks and performance.

## Features
- Add student name and marks
- Automatically calculate Pass/Fail
- Display all student records
- Show average marks
- Show topper details

## Technologies Used
- Python
- Flask
- SQLite
- HTML

## How to Run

1. Install Flask:
   pip install flask

2. Run the application:
   python app.py

3. Open in browser:
   http://127.0.0.1:5000
